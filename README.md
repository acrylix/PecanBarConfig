# Features

| Features     | Requirement
| ------------ | ----------------------
| WM info      | chunkwm + chunkc
| CPU Temp     | `scripts/osx-cpu-temp`     
| network info | `ifstat`
| battery info | -

- Make sure Übersicht here http://tracesof.net/uebersicht/

- clone into `Übersicht/widgets`

- refresh widgets
